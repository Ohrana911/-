## Немного о проекте

Проект представляет собой систему для классификации дерматоскопических изображений на два класса:
- **0 - Доброкачественные** новообразования (melanocytic nevus, benign keratosis)
- **1 - Опасные/Злокачественные** новообразования (melanoma, basal cell carcinoma, actinic keratosis)

**Цель**: Создание прототипа системы мониторинга состояния кожи, которая может использоваться как вспомогательный, но не предназначенный для точной мед. диагностики инструмент для повышения осведомленности о рисках кожных заболеваний.

## Связь с мониторингом экосистемы

Данный проект реализует подход **"мониторинга здоровья человеческой популяции как части экосистемы"**:

**Заболеваемость меланомой напрямую зависит от таких факторов как:**
- Изменение озонового слоя вызывает повышение ультрафиолетового излучения
- Глобальное потепление большая проблема на текущий момент, солнечная активность повышается, что влечет увеличение кол-ва солнечных дней
- Загрязнение воздуха оказывает канцерогенное воздействие на кожный покров

## Архитектура

### Модель
- **Базовая модель**: Vision Transformer (ViT-Base) предобученная на ImageNet-21k
- **Архитектура**: Трансформер для классификации изображений
- **Размер входного изображения**: 224×224 пикселей
- **Количество классов**: 2 бинарная классификация

### Технологический стек
- **Фреймворк**: PyTorch 2.0+
- **Обработка изображений**: TorchVision, OpenCV
- **Трансформеры**: Hugging Face Transformers
- **Анализ данных**: Pandas, NumPy, Scikit-learn
- **Визуализация**: Matplotlib, Seaborn
- **Инфраструктура**: Google Colab, GitHub

### Матрица ошибок

| | Предсказано: Доброкачественная | Предсказано: Опасная |
|------------------------|--------------------------------|----------------------|
| **Истинно: Доброкачественная** | 252 (84%) | 48 (16%) |
| **Истинно: Опасная** | 44 (15%) | 250 (85%) |

**Объем тестовой выборки:** 594 изображения

### Анализ ошибок:
- **16% ложных срабатываний**: Доброкачественные → Опасные
- **15% пропусков**: Опасные → Доброкачественные
- **Равномерное распределение ошибок**: модель не смещена к одному классу

Итог:
Модель устойчива и хорошо различает классы.
