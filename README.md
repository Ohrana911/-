**Проектная практика МИФИ Урамаев**

Беру тему: "Классификация типов новообразований на коже человека". Это относится к теме экологии напрямую, так как заболеваемость меланомой напрямую зависит от экологических факторов. К таким факторам относится учащение солнечных дней как пример.

Данный проект посвящен разработке системы мониторинга здоровья человеческой популяции как части экосистемы.
